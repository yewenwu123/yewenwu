# Harbor Frame Sourcing

Static website for an Australia-focused eyewear sourcing business.

## Files you will edit most often

- `data/site-config.js`: your name, contact details, about text, form endpoint
- `data/products.js`: product/style cards
- `assets/images/`: life photos and style photos

## Free hosting

This project is designed for GitHub Pages. You can publish it directly from the repository root.

## Form setup

The consultation form needs a form endpoint. Add it in `data/site-config.js`.

Example services:

- Formspree
- FormSubmit

## Local preview

Open `index.html` in a browser, or use a simple static server if you prefer.
